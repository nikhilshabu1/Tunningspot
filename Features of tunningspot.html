Features of Tunningspot

  • Customizable log-on and log-off pages
  • Option to redirect customers to a specific landing page (advertisement option)
  • SSL user Login page/management page
  • Web-Based login
  • NAT (RFC 1631) / DHCP Server
  • Routing functionality
  • RADIUS AAA support (RFC 2865, 2866)
  • Built-in firewall
  • Load balancer/failover
  • Virtual LAN creation
  • Multiple WAN/LAN option
  • Separate captive portal for each LAN
  • Redirection URL feature
  • User / MAC / PIN based authentication
  • Option to bypass login page for authenticated users
  • User accounting and authentication
  • Multiple custom plans based on time, data or bandwidth
  • Internet access time feature /Bandwidth control / Access control based on time & data
  • Multiple coupon creation and printing options
  • Option to do content filtering (sites & keyword)
  • Option to do advanced content filtering (HTTPS sites and category based)
